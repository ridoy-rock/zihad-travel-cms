== Description ==

Zihad Travel CMS is a professional travel content management system for WordPress. It provides everything a travel agency needs:

* Tour packages with itineraries, pricing and galleries
* Visa services with types and requirements
* Country destination guides
* Booking and enquiry management
* Elementor-compatible, translation-ready, Bootstrap 5 frontend

== Installation ==

1. Upload the plugin to `/wp-content/plugins/zihad-travel-cms`.
2. Activate it through the Plugins screen.
3. Configure it under **Travel CMS → Settings**.
